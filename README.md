# Biosensors for Early Detection of Neurodegenerative Diseases

## Project Overview
This repository contains the research report and presentation developed for **Eureka 8.0 (SVNIT)** under the domain of Biochemical Engineering for Health and Environment. 

The project proposes a non-invasive, next-generation electrochemical biosensor platform utilizing Graphene Derivative Materials (GDMs) to achieve ultra-sensitive, early-stage detection of biomarkers for Alzheimer's and Parkinson's diseases.

## Contents
* [Research Report (PDF)](Biosensor_ND_Report.pdf) - Full technical review and analytical framework.

## Core Core Concepts Explored
* **Biochemical Engineering:** Process modeling for decentralized point-of-care diagnostics.
* **Surface Chemistry:** Electrode modification using nanomaterials for enhanced electron transport.
* **Data & AI/ML Pipelines:** Conceptual frameworks for leveraging machine learning classification algorithms to process multiplexed biological signals.
